# [9] Bootstrap

## Apa itu Bootstrap?

Bootstrap adalah CSS framework yang dapat digunakan untuk mempermudah dalam pembuatan tampilan suatu halaman web. Bootstrap menyediakan fitur untuk mempercepat styling yang dilakukan pada elemen elemen HTML yang dibuat.

## Keunggulan Bootstrap

Bootstrap memiliki keunggulan yang akan mempermudah front-end development.

- Cara install dan pemasangan bootstrap pada project sangat mudah (dapat melalui NPM atau CDN)
- Dapat membuat web responsive dengan lebih mudah (dapat memanfaatkan grid system yang sudah disediakan).
- Memiliki banyak komponen yang siap digunakan sehingga tidak perlu membuat dari awal.

## Penggunaan Bootstrap

Bootstrap perlu diinstall atau dipasangkan dulu ke dalam project sebelum digunakan (dapat menggunakan NPM atau CDN). Setelah sudah terpasang dengan benar, bootstrap dapat digunakan untuk styling elemen-elemen HTML dengan cara memberikan nama-nama class dalam bootstrap pada elemen elemen tersebut.
Misalnya jika ingin memberi style pada elemen Button:

```html
<button type="button" class="btn btn-primary">Button</button>
```
